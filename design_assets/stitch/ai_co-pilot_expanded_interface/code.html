Here is the implementation using ReduceLROnPlateau. This will dynamically adjust the learning rate when the validation loss plateaus.

scheduler_config.py 
import torch.optim as optim

# Define optimizer
optimizer = optim.Adam(model.parameters(), lr=1e-3)

# Define adaptive scheduler
scheduler = optim.lr_scheduler.ReduceLROnPlateau(
    optimizer, 
    mode='min', 
    factor=0.1, 
    patience=5, 
    verbose=True
)

# In training loop
scheduler.step(val_loss)
